transition 属性

属性详解：https://developer.mozilla.org/zh-CN/docs/Web/CSS/transition

这里值得一提的是，在设置

div {

  some code...

  &.active {
    position: relative;
    top: 5px;

    transition: top .3s;      
  }  
}

时，如果没有之前声明position: relative，这里的top渐变会失效。即使transition: all .3s也没办法实现。

说明 transition-property 属性值不能是 position
